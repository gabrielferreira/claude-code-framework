# Migration v2.39.0 → v2.40.0

> Data: 2026-04-15
> Tipo: minor

## Resumo

Nova skill `/onboarding` que gera guia contextualizado do fluxo de trabalho para devs novos no projeto. Lê artefatos reais e monta output no chat com comandos, skills instaladas, agents e fluxo adaptado.

## Pre-requisitos

- Versao atual do projeto: v2.39.0
- Se estiver em versao anterior, aplique as migrations anteriores primeiro

## Mudancas

### 🆕 Arquivos novos

#### 1. `.claude/skills/onboarding/SKILL.md` (novo)

**O que e:** Slash command `/onboarding` — gera guia contextualizado do fluxo de trabalho lendo CLAUDE.md, PROJECT_CONTEXT.md, SETUP_REPORT.md e SKILLS_MAP.

**Relevante se:** todos os projetos. Especialmente util para onboarding de devs novos.

**Como instalar:** Copiar do framework.

```bash
mkdir -p .claude/skills/onboarding
cp {framework}/skills/onboarding/SKILL.md .claude/skills/onboarding/SKILL.md
```

**Se decidir nao instalar:** nenhum impacto — e opcional. Devs podem consultar QUICK_START e SPEC_DRIVEN_GUIDE diretamente.
