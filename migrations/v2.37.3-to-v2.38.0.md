# Migration v2.37.3 → v2.38.0

> Data: 2026-04-10
> Tipo: minor

## Resumo

Nova skill `/pr` que preenche template de PR com contexto de spec + diff e abre via `gh pr create`. Distribuição de `.github/pull_request_template.md` via setup-framework.

## Pre-requisitos

- Versao atual do projeto: v2.37.3
- Se estiver em versao anterior, aplique as migrations anteriores primeiro

## Mudancas

### 🆕 Arquivos novos

#### 1. `.claude/skills/pr/SKILL.md` (novo)

**O que e:** Skill `/pr` — slash command que detecta spec associada à branch, lê o diff, preenche o template de PR com contexto estruturado (O que muda, Por quê, Como testar, Checklist) e abre via `gh pr create` após confirmação.

**Relevante se:** Sempre — qualquer projeto que abre PRs se beneficia.

**Como instalar:** Copiar `skills/pr/SKILL.md` para `.claude/skills/pr/SKILL.md`.

```bash
mkdir -p .claude/skills/pr
cp {framework_path}/skills/pr/SKILL.md .claude/skills/pr/SKILL.md
```

---

#### 2. `.github/pull_request_template.md` (novo)

**O que e:** Template de PR padrão do framework com seções: O que muda, Por quê, Como testar, Checklist. Usado pela skill `/pr` e pelo GitHub nativamente em todo novo PR.

**Relevante se:** Sempre — se o projeto já tem template próprio, manter o existente.

**Como instalar:** Copiar `{framework_path}/.github/pull_request_template.md` para `.github/pull_request_template.md`. Se já existe um template, avaliar se quer substituir ou manter o existente.

```bash
mkdir -p .github
cp {framework_path}/.github/pull_request_template.md .github/pull_request_template.md
```

---
