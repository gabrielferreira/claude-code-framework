# Migration v2.17.0 → v2.17.1

> Data: 2026-04-07
> Tipo: patch

## Resumo

Correcoes para modo Notion: setup e update agora filtram artefatos locais desnecessarios (backlog.md, TEMPLATE.md, etc.) e o update remove ativamente esses arquivos se existirem. Setup tambem passou a ler o VERSION do framework para evitar framework-tags com v0.0.0.

## Pre-requisitos

- Versao atual do projeto: v2.17.0
- Se estiver em versao anterior, aplique as migrations anteriores primeiro
