# MO7 — i18n das skills

**Descrição:** suporte a múltiplos idiomas nas skills — traduzir instruções para inglês, espanhol, etc.

**Descartado em:** 2026-04-09

**Motivo:** skills são instruções para o Claude (LLM), não UI para usuário final. O Claude processa qualquer idioma sem tradução de skill — a i18n não gera valor real aqui.
