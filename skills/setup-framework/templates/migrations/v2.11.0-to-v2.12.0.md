### Ciclo de vida dos status

| Status | Significa | Pode implementar? |

### 👀 Manual (decisao humana)

#### 8. `CLAUDE.md`

**O que mudou no template (7 alteracoes):**

**a) Nova secao "Output — concisao obrigatoria" (apos titulo, antes de "O que e este projeto"):**

```markdown
## Output — concisão obrigatória

Toda saída de texto deve ser curta e direta. Verbosidade é custo, não qualidade.

- **Sem preâmbulos.** Nunca "Let me first…". Substituir por verbo + sujeito: `Lendo rotas…`
- **Status em 1 linha.** Descrever o que está fazendo, não por quê.
- **Sem conclusões óbvias.** Eliminar "Now I have enough context".
- **Reports de sub-agents:** finding + arquivo + linha. Sem intro.
- **Erros/blockers:** direto. `FAIL: auth.spec.js:42 — timeout`
- **Respostas a perguntas:** responder, não narrar.
```

**b) Nova secao "TDD obrigatorio" (apos "Specs e Requisitos"):**

```markdown
## ⛔ TDD obrigatório

Testes escritos ANTES da implementação. Ciclo: red → green → refactor.
Exceção: bug urgente em produção (<30min).
```

**c) Skills convertidas de lista para tabela com coluna "Obrigatorio?"**

Substituir a lista numerada por tabela. Adicionar skill `execution-plan` como item 2.

**d) Agents convertidos de lista para tabela com modelo, trigger e obrigatoriedade**

Adicionar 3 novos agents: `dx-audit`, `performance-audit`, `infra-audit`.

**e) Nova secao "Execucao por agents — orquestracao" (apos Agents):**

Checkpoint obrigatorio, paralelismo dentro de item, regras de delegacao.

**f) Checklist pre-commit expandido (de 3 para 9 itens):**

Adicionar: testes, coverage, verify.sh, verificacao manual, spec, parcial, regra nova, E2E, docs.

**g) Secao "Testes" renomeada para "Testes e coverage" com tabela de targets:**

```markdown
| Camada | Statements | Branches |
|--------|-----------|----------|
| Backend | 100% | ≥95% |
| Frontend | 100% | ≥90% |
| E2E | Fluxos user-facing | — |
```

**Sugestao:** Aplicar todas as mudancas — sao melhorias de processo validadas em producao. Se o projeto nao usa TDD, ajustar a secao b) para "testes obrigatorios".
